# VS Code Key shortcuts

## Windows

- Close a tab:
```sh
  Ctrl + W
```

- Comment a line/ block of code:
```sh
  Ctrl + /
```

- Duplicate a line of code:
```sh
  Ctrl + D
```

- Using the find feature in your file:
```sh
  Ctrl + F
```

- Use the find feature in your project:
```sh
  Ctrl + Shift + F
```

- Open any file instantly:
```sh
  Ctrl + P
```

- Edit multiple lines at once:
```sh
  Alt + Left_Click
```

- Indent a line/ block of code:
```sh
  Tab 
```

- Unindent a line/ block of code:
```sh
  Shift + Tab
```
